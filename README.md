
# 📊 Adani Stock Data Analysis Dashboard

## 📌 Project Overview
This project is a stock market dashboard focused on Adani stock data.  
The dashboard is designed to help understand stock performance, trends, and trading behavior in a simple and visual way.


## ❗ Problem Statement
Stock market data is large and difficult to understand in raw format.  
People find it hard to quickly identify trends, price movements, and performance.

This project solves this problem by:
- Converting raw data into visual charts
- Making analysis easy and fast
- Helping users understand stock behavior clearly


## 🎯 Objective
- To analyze stock price trends
- To track performance over time
- To simplify decision-making using visuals


## 📊 Dashboard Features

### 🔹 KPI Cards
- 52 Week High
- 52 Week Low
- Close Total
- Open Total
- Volume Total

### 🔹 Charts Included
- Quarterly Aggregate → shows performance across quarters
- Daily Totals by Date → shows stock movement over time
- Monthly Totals → compares monthly performance
- Volume by Month → shows trading activity
- 52 Week High vs Low → comparison chart

### 🔹 Filters (Slicers)
- Day filter
- Month filter
- Year filter

These filters allow users to interact with the dashboard easily.
